This driver simulates a USB HID joystick device with up to 64 buttons, 8 axes and 4 POV (Hat) switches.


Dependencies
============
This driver depends on:

* `Adafruit CircuitPython <https://github.com/adafruit/circuitpython>`_ (>= 7.0.0)
